## Installation

Use the package manager [npm](https://www.npmjs.com/) to install foobar.

Install the package in Backend directory and client directory inside frontend directory

```bash
npm install
```

To Run the server

```bash
npx nodemon app.js
```

To Run the client

```bash
npm run dev
```

To Run the Admin

```bash
npm run dev
```
